My Portofolio

YelpCamp

YelpCamp is a website that is very useful for users who want to find nearby campground, see the campground reviews, and rating. This website includes the web API so user can see the precise location of the campground. Also, users who own a campground can add it to the YelpCamp. This project is developed during the Web Developer Bootcamp in Udemy.

Tools used to develop YelpCamp are: HTML, CSS, Javascript, Bootstrap, Node.js, Express.js, MongoDB, Passport, Cloudinary, etc.

Dicer!

Dicer! is a dice game for two players. The gameplay is relatively simple, the first player to reach 100 points is the winner. New game feature is also provided in case the user wants to restart the game.

Tools used to develop Dicer! are: HTML, CSS, Javascript.
